API PARA LA GESTIÓN DE LOS PSM DE LOS COLEGIOS SALESIANOS

DESCRIPCIÓN

Esta API gestiona los usuarios de la misma según los roles de USUARIO, ADMIN y SUPERADMIN

El usuario básico podrá

  • Crear un nuevo PSM procesando un archivo CSV para introducir los datos.
  • Eliminar un PSM con todos sus datos.
  • Listar los PSM de un Curso Académico para un Colegio
  • Listar los valores de PSM de varios Cursos Académicos
  • Ver analítica y gráficamente la tendencia de los valores de un indicador a lo largo de varios Cursos Académicos

El usuario administrador podrá:

  • Todo lo que hace un Usuario básico.
  • Crear usuarios básicos del Colegio que administra.
  • Importar la estructura del Colegio para un determinado curso desde un fichero CSV.
  • Gestionar las Etapas y subetapas de un Colegio.
  • Gestionar los Cursos de una Etapa.
  • Gestionar las Unidades de un Colegio.
  • Gestionar los Puntos de Control de un Colegio para un Curso Académico.
  • Configurar de forma rápida la estructura de un Colegio para un Curso Académico.
  • Configurar de forma rápida los Procesos e Indicadores de un Colegio para un Curso Académico.

El usuario SuperAdministrador podrá:

  • Todo lo que hace un Usuario básico y un Administrador.
  • Crear Administradores de cada Colegio.
  • Crear Cursos Académicos.
  • Eliminar Cursos Académicos si no tienen datos asociados.
  • Gestionar los Procesos de un Colegio.
  • Gestionar los indicadores de un Proceso.
